# Minimal Planner

A minimal dashboard for managing reminders and notes.

## Features

- **Reminders**: Add reminders with dates and types (Deadline, Test, Other).
- **Notes**: simple text notes for quick thoughts.
- **Local Storage**: Data is persisted in your browser's local storage.
- **Minimal UI**: Clean, distraction-free design.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
